# Badge

```javascript
import { Badge } from 'mfcl'
```

## Description

Badges can be added to show small, simple details of information.

## Properties

<Props of={Badge} />

## Badge Types

A badge can be of two types, primary and secondary.

<Playground>
  <Badge>Save $100</Badge>
  <span style={{ paddingLeft: '5px' }} />
  <Badge type="secondary">Save $100</Badge>
</Playground>

## Adding Links

A link can also be provided to the badge by using the href prop.

<Playground>
  <Badge href="https://www.youtube.com/watch?v=oHg5SJYRHA0">Click Me</Badge>
  <span style={{ paddingLeft: '5px' }} />
  <Badge href="https://www.youtube.com/watch?v=oHg5SJYRHA0" type="secondary">
    Click Me
  </Badge>
</Playground>

## Custom Width and Height

You can also set a custom width/height using the provided props as well.

<Playground>
  <Badge width="200px">WideGuy</Badge>
  <span style={{ paddingLeft: '5px' }} />
  <Badge height="50px" type="secondary">
    TallMan
  </Badge>
</Playground>
