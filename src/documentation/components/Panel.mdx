# Panel

```javascript
import Panel from 'mfcl/dist/Panel'
```

## Description

Panels group related content and may contain actions about that content

## Panel Properties

<Props of={Panel} />

## Panel Item Properties

<Props of={PanelItem} />

## Basic Usage and variations

Panels come in two variations and house panel items within them

<Playground>
  <Panel>
    <PanelItem>Content</PanelItem>
  </Panel>
  <br />
  <Panel rounded>
    <PanelItem>Content</PanelItem>
  </Panel>
</Playground>

## Headers and Footers

Special panel item types can be designated by the type property

<Playground>
  <Panel>
    <PanelItem type="header">Header</PanelItem>
    <PanelItem type="body">Body</PanelItem>
    <PanelItem type="link" href="/components-flag">
      Link
    </PanelItem>
    <PanelItem type="footer">Footer</PanelItem>
  </Panel>
</Playground>

## Custom Styling

Apply custom styling to both panel and panel item components

<Playground>
  <Panel rounded customStyling={{ color: 'red' }}>
    <PanelItem type="header" customStyling={{ textAlign: 'center' }}>
      Centered Header
    </PanelItem>
    <PanelItem>Some information</PanelItem>
    <PanelItem type="footer" customStyling={{ backgroundColor: 'grey', color: 'white' }}>
      footer
    </PanelItem>
  </Panel>
</Playground>
