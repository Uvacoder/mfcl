# Input

```javascript
import { Input } from 'mfcl'
```

## Description

Inputs accept user entry data. Inputs use labels and sometimes icons to communicate the input field that will accept data when a user submits them.

## Properties

<Props of={Input} />

## Basic Usage

An Input field can show a user is currently interacting with it through focus styling.  
Generic attributes for input elements can be applied.

<Playground>
  <Input type="text" label="Default" name="name" focus />
</Playground>

## Size

You can choose between medium and large input field sizes, the default being large.

<Playground>
  <Input type="text" size="md" label="md" />
  <br />
  <Input type="text" size="lg" label="lg" />
</Playground>

## Required or Disabled

Make input required or disabled.  
Disabled attribute cascades to inline button.

<Playground>
  <Input type="text" required label="requied" />
  <br />
  <Input type="text" disabled label="disabled" />
</Playground>

## Footnote

Add a small description below the input field.

<Playground>
  <Input inputMessage="This is an info message" label="info" />
  <br />
  <Input error inputMessage="This is an error message" label="error" />
  <br />
  <Input disabled error inputMessage="This is an error message" />
</Playground>
