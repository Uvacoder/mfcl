# Price

```javascript
import Price from 'mfcl/Price'
```

## Description

The Price component renders and formats the pricing to USD. The optional props allow for further customization on how
the Prices render and look.

## Properties

<Props of={Price} />

## Basic Usage

The Price component requires the Price prop to render.

<Playground>
  <Price price={[3500]} />
</Playground>

## Discounted Price

Allows a price markdown below the discount price

<Playground>
  <Price discountPrice={[1400]} price={[3500]} />
</Playground>

### Optional Props

A price can be discounted without the price markdown below it by adding the optional Discount prop. This also works with the Text and Divider prop. The content can also be center aligned by using the Center prop.

<Playground>
  <Price discount price={[3500]} />
  <Price text="Queen For" discount price={[3500]} />
  <Price text="Queen" divider discount price={[3500]} />
  <Price center text="Queen" divider discountPrice={[1400]} price={[3500]} />
  <Price right discountPercentage discountPrice={[1400]} price={[3500]} />
</Playground>
