# Price

```javascript
import Price from 'mfcl/Price'
```

## Description

The Price component renders and formats the pricing to USD. The optional props allow for further customization on how
the Prices render and look

## Properties

<Props of={Price} />

## Basic Usage

The Price component requires the Price prop to render

<Playground>
  <Price price={[3500]} />
  <Price price={[3500, 5000]} />
</Playground>

## Alignment

The Price component can be aligned horizontally

<Playground>
  <Price price={[3500]} />
  <Price price={[3500]} center />
  <Price price={[3500]} right />
</Playground>

## Text

The Price component can have on either side of the price, with or without a divider

<Playground>
  <Price price={[3500]} text="Queen" />
  <Price price={[3500]} text="Queen" divider />
  <Price price={[3500]} rightText="/mo" />
</Playground>

## Discounts

The Price component can be provided another array of prices indicating original pricing before a discount along with the percentage if desired

<Playground>
  <Price price={[1400, 2000]} originalPrice={[3500, 5000]} discountPercentage />
</Playground>

## Precision

The Price component can show decimals which can be aligned vertically

<Playground>
  <Price originalPrice={[3500]} price={[1400, 2000]} decimalsTop />
  <Price originalPrice={[3500, 5000]} price={[2000]} decimalsBottom />
</Playground>
