# Tabs

```javascript
import { Tabs } from 'mfcl'
```

## Description

Tabs allow you to switch between different views of related information.

## Properties

<Props of={Tabs} />

## Basic Usage

<Playground>
  <Tabs name="default" items={tabData} />
</Playground>

## Tabs Side

<Playground>
  <Tabs name="side" position="left" items={tabData} />
</Playground>

## Controlled or Uncontrolled

<Playground>
  <Component initialState={{ activeTab: 1 }}>
    {({ setState, state }) => (
      <>
        <Tabs
          name="controlled"
          onChange={(activeTab) => setState({ activeTab })}
          value={state.activeTab}
          items={tabData}
        />
        <Tabs name="uncontrolled" defaultValue={3} items={tabData} />
      </>
    )}
  </Component>
</Playground>
