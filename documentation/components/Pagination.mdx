# Pagination

```javascript
import Pagination from 'mfcl/Pagination'
```

## Description

Pagination allows you to paginate through a list of items.

## Properties

<Props of={Pagination} />

## Basic Usage

<Playground>
  <Row>
    <Pagination totalPages={10} />

</Playground>
