## Basic Usage

NOTE: DO NOT add "Component" as a wrapper. That is only for documentation. Use React state/context to manage open/close

<Playground>
  <Component initialState={{ isOpen: false }}>
    {({ setState, state }) => {
      return (
        <div>
          <Modal
            header="Basic Modal"
            subheader="Subheader Details"
            isOpen={state.isOpen}
            onClose={(isOpen) => setState({ isOpen: false })}
          >
            {mattressIpsum(3)}
          </Modal>
          <Button btnType="primary" onClick={() => setState({ isOpen: !state.isOpen })} size="lg">
            Open Modal
          </Button>
        </div>
      )
    }}
  </Component>
