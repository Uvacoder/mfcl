# HeroBanner

```javascript
import HeroBanner from 'mfcl/HeroBanner'
```

## Description

The hero banner allows you to display important information alongside images and buttons.

## Properties

<Props of={HeroBanner} />

## Basic Usage

<Playground>
  <HeroBanner
    brandLogo={<SleepysBrand />}
    heroBannerStyle={{ backgroundColor: '#0A1232' }}
    imageSrc={SleepyImg}
    imageAlt="i am ironman"
    title="Achieve Better Sleep"
    body="Sign up to get better insider tips & expert advice in your inbox."
    btnText="Vist Sleep.com"
    btnLink="http://localhost:4000/components-heroBanner"
  />
</Playground>

## Single or Dual Hero banner

- You can make this a single banner without the dual sides by simply removing the image. The background color of
  the left tile can be changed by passing the **bgColor** prop.

<Playground>
  <HeroBanner

## Add/Remove a Button

- If the **btnText** is empty, or doesn't exist, the component won't render a button.

- You can change the image by providing the **imageSrc** prop an image string.
